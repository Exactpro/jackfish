Via using special frame around necessary part of text You can set format for text.
All patterns contains &#123;&#123; and some special symbol, text, again same special symbol and &#125;&#125;.
Different symbols set different format for determined text. For example, if You use &#123;&#123;&#42; and &#42;&#125;&#125; frame around text 'Hello, world!' You will get bolder text '{{*Hello, world!*}}'.
In the table You can find all available symbols and their functions with examples.
